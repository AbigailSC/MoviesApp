# MoviesApp

## Página creada con React, Redux toolkit y TailwindCss, podes ver las ultimas peliculas en cartelera y mirar sus detalles (como la puntuación por los usuarios, los generos, los comentarios, etc), para más información acerca de la pelicula tenes un enlace directo al sitio de IMDB en donde encontraras más información.

## Ademas posee busqueda por nombre, la barra de busquedas no se activara hasta que ingrese algun texto y en caso de no encontrar el nombre que ingresaste te mostrara un texto con el error.

## Por ultimo tiene redireccionamiento a la ruta de error404 en caso de que intentes entrar a alguna otra ruta

## La pagina fue desarrollada con el principio de Mobile First y tiene modo oscuro.

## Instalación rapida

### 1. Clona el repositorio

### 2. Copia y pega los siguientes comandos en la consola de Visual Studio

### 3. `npm install`

### 4. `npm run dev`

### 5. Abri http://127.0.0.1:5173/ en tu navegador

### 6. Registrate en The Movie Database(TMDB) para obtener la api key de la api, una vez que la tengas crea un archivo .env en la raiz de la carpeta moviesApp y crea la siguiente variable en tu archivo .env: VITE_APP_API_KEY = "tu apikey"
